# Premium Admin Dashboard

แดชบอร์ดสำหรับจัดการคำสั่งซื้อ สร้างด้วย React + Vite + Tailwind CSS
โทนสีน้ำเงิน-ขาว สไตล์มินิมอลแบบพรีเมียม ไม่มีอิโมจิในส่วนของอินเทอร์เฟซ

## เริ่มต้นใช้งาน

```bash
npm install
npm run dev
```

แอปจะรันที่ `http://localhost:5173`

## โครงสร้างโปรเจกต์

```
src/
├── components/
│   ├── Sidebar.jsx       เมนูนำทางด้านซ้าย (โลโก้ + เมนู + โปรไฟล์)
│   ├── Topbar.jsx        แถบบนสุด (ค้นหา + แจ้งเตือน + โปรไฟล์)
│   ├── StatCard.jsx      การ์ดสรุปข้อมูลตัวเลขสำคัญ
│   ├── StatusBadge.jsx   ป้ายแสดงสถานะคำสั่งซื้อ
│   └── DataTable.jsx     ตารางข้อมูล (ค้นหา / กรอง / เรียงลำดับ / pagination)
├── data/
│   └── orders.js         ข้อมูลตัวอย่าง — แทนที่ด้วย API จริงได้ทันที
├── App.jsx                ประกอบหน้าแดชบอร์ดจากทุก component
├── main.jsx
└── index.css
```

## Design tokens (กำหนดไว้ใน tailwind.config.js)

- `navy-950` `#0A1F44` — พื้นหลัง sidebar
- `accent-500` `#2655FF` — สีหลักสำหรับปุ่ม / สถานะ active / ลิงก์
- `surface` `#F4F7FC` — พื้นหลังเนื้อหาหลัก
- `surface-card` `#FFFFFF` — พื้นหลังการ์ดและตาราง
- `ink-900` `#0B1220` — ข้อความหลัก

ฟอนต์: **Space Grotesk** (หัวข้อ/ตัวเลขใหญ่), **Inter** (เนื้อหาทั่วไป),
**JetBrains Mono** (รหัสคำสั่งซื้อ ตัวเลข และยอดเงินในตาราง เพื่อให้ตัวเลขเรียงตรงคอลัมน์)

## นำไปต่อยอด

- เชื่อมต่อ `src/data/orders.js` เข้ากับ API จริงแทนข้อมูลตัวอย่าง
- เพิ่ม React Router หากต้องการให้เมนูใน Sidebar เปลี่ยนหน้าจริง
- คอมโพเนนต์ทั้งหมดรับ props แยกจากข้อมูล จึงนำไปใช้กับชุดข้อมูลอื่น (ลูกค้า, สินค้า ฯลฯ) ได้โดยปรับ column เพียงเล็กน้อย
